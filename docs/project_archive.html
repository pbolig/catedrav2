🛠️ Instrucciones de Instalación

Paso 1: Descomprimir proyecto 
unzip catedra-manager-nodejs.zip
cd catedra-manager
Paso 2: Instalar dependencias 
npm install
Paso 3: Configurar MySQL 
# Crear base de datos
mysql -u root -p
CREATE DATABASE catedra_db CHARACTER SET utf8mb4;
Paso 4: Configurar variables de entorno 
cp .env.example .env
# Editar .env con tus datos de MySQL
Paso 5: Inicializar base de datos 
npm run setup-db
npm run seed-db
Paso 6: Iniciar aplicación 
npm run dev
